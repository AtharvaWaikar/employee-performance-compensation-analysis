# 📊 Employee Performance & Compensation Analysis
## 📌 Project Overview

This project analyzes Employee Performance and Compensation (EPC) data to identify performance trends, salary patterns, and workforce insights. The objective is to explore how performance metrics relate to compensation using structured HR datasets.

## 🎯 Objectives
Clean and preprocess employee data
Analyze performance ratings and compensation distribution
Generate summary statistics
Extract actionable HR insights

## 🛠️ Tools & Technologies
1] Python\
2] Pandas\
3] NumPy\
4] Jupyter Notebook\
5] CSV Data Files

## 📂 Project Files
1] Epc.ipynb – Main analysis notebook\
2] cleaned_employee_data.csv – Processed dataset\
3] summary_statistics.csv – Generated statistical insights

## 📈 Key Analysis Performed
1] Data cleaning and preprocessing\
2] Descriptive statistics generation\
3] Performance vs compensation trend analysis\
4] Insight extraction from structured HR data

## 🚀 Project Outcome
The analysis provides meaningful insights into employee performance patterns and compensation structure, supporting data-driven workforce decisions.
